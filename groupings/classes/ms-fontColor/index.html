    Samples by Class: ms-fontColor-*¶

    The ms-fontColor-* classes apply a font color. Several standard, shared, theme, neutral, and more are included.

    Samples are grouped by classes used. Expand to see the related samples.

    Complimentary Classes

    These classes compliment the ms-borderColor-* and ms-bgColor-* classes and except where noted (on those pages) contain equivalent colors.
