# Samples by Class: ms-font-*

The `ms-font-*` classes apply various combinations of text **size** and **font weight**. To apply just size or just weight use the [ms-fontSize-*](./ms-fontSize.md) and [ms-fontWeight-*](./ms-fontWeight.md) classes respectively.

Samples are grouped by classes used and listed from smallest to biggest. Expand to see the related samples.
