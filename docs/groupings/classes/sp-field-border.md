# Samples by Class: sp-field-border*

The `sp-field-border*` classes ending in `Regular`, `Semibold`, and `Bold` set a **border-width** to one or more sides (`1px`, `2px`, and `3px` respectively). The `sp-field-border*` classes ending in `Solid`, `Dashed`, and `Dotted` set a **border-style** to one or more sides.

Samples are grouped by classes used. Expand to see the related samples.
