# Person Facepile Rotator

## Summary
This sample implements a multi-person facepile formatting layout. In standard facepiles, only the first three members are visible while other members remain hidden behind the overflow count. This formatter introduces "face fairness" by establishing a rotation seed based on the current system minute. As users refresh the page, different consecutive groupings of up to three members cycle into view.

![screenshot of the sample](./assets/screenshot.png)

## Column Requirements
This formatting must be applied to a multi-selection Person column.
